# (C# 코딩) 에코 메신저
## 개요
- C# 프로그래밍 학습
- 핵심기능:
1. 사용자가 입력한 문자열을 리스트박스에 전송
2. 공백이나 NULL의 경우 리스트박스에 전송되지 않음
3. 전송된 문자열은 전송한 시간이 표시됨(타임스탬프)
4. 삭제 버튼으로 개별 메세지 삭제 가능
5. 대화내용 전체삭제 버튼으로 초기화
6. 대화 메세지가 50자를 넘기면 오류 메세지 출력

- 화면구성: 리스트박스, 텍스트박스(메세지전송창), 전송버튼, 메세지카운터

## 배운 내용
-버튼 클릭 외에 키보드 컨트롤로 버튼을 컨트롤을 구현하는 방법
-텍스트 박스에서 입력한 문자열의 앞 뒤 공백을 제거하는 방법
-리스트 박스에서 선택한 아이템을 삭제하는 방법
-리스트 박스의 아이템을 초기화하는 방법
-메세지 카운터를 구현하는 방법
-메세지의 길이를 체크하여 오류 메세지를 출력하는 방법
-메세지에 타임스탬프를 추가하는 방법
-예외에 대한 오류 메세지 표기 방법
